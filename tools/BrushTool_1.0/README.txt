Installation:
    • Copy the brush and packages folders to the Houdini user preferences folder:
      Houdini preferences folder paths on different platforms:
        Windows: C:\Users\user\Documents\houdini19.5
        Linux: /home/user/houdini19.5
        MacOS: /Users/user/Library/Preferences/houdini/19.5

    • Run Houdini
    • Add the Brush shelf. To set hotkey click the shelf tool with Ctrl+Alt+Shift+LMB
